    Wat heb je zoal gemaakt en wat kun je laten zien?

    Tickomatic 2000

    De Tickomatic 2000 is een webapp waarmee je je typsnelheid (het gemiddelde aantal woorden dat je per minuut en per seconde typt) kunt bepalen. Dit doe je door de woorden uit de woordenlijst die je selecteert op de juiste manier over te typen. Vervolgens doet de Tickomatic 2000 de rest. Ik heb deze app gemaakt omdat ik graag mijn typsnelheid wil blijven verbeteren.

    HTML5 / CSS3 / SCSS / Bootstrap / JavaScript / jQuery / React / React-Transition-Group
    Gehoste versie Code Documentatie
    Daily Programming Challenges

    De Daily Programming Challenges webapp toont je drie verschillende opdrachten die je kunt doen ter oefening. Deze oefeningen worden willekeurig uit een voorafgemaakte lijst geselecteerd en gecombineerd met een x aantal keer dat de oefening dient te worden uitgevoerd. Dit zijn vooral oefeningen die ik vaak wil blijven herhalen tijdens het (leren) programmeren.

    HTML5 / CSS3 / SCSS / Bootstrap / Gulp
    Gehoste versie Code Documentatie
    Pokémon Team Analysis

    De Pokémon Team Analysis app is een webapp waarmee je kunt zien welk Pokémon-type effectief is tegen jouw team en tegen welk Pokémon-type jouw team zelf effectief is. Je geeft op welke Pokémon je in jouw team hebt zitten en de Team Analysis app laat je precies zien hoe het gesteld is met jouw type-effectiveness.

    HTML5 / CSS3 / SCSS / Bootstrap / JavaScript / jQuery / React / Redux / Axios / Thunk / React-Transition-Group
    Gehoste versie Code Documentatie
    Sights To See On Sicily

    Sights To See On Sicily is een multi-page website waarop je de hotspots van Sicilië kunt ontdekken door middel van informatie en foto's. Dit project vloeit voort uit mijn liefde voor Italië: de natuur, de cultuur en de sfeer.

    HTML5 / CSS3 / SCSS / Bootstrap / JavaScript / jQuery / React / React Router / React-Transition-Group
